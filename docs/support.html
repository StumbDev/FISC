Frequently Asked Questions

Q: How do I report a bug?

A: Use the form above and select "Bug Report" as the subject. Please include:

  • FISC-V version
  • Your operating system
  • Steps to reproduce the issue
  • Expected vs actual behavior

Q: Where can I find configuration examples?

A: Check our documentation page for various configuration examples.

Q: How do I request a new feature?

A: Use the form above or create an issue on our GitHub page.
